# Dependencies

- nmap
- tabulate
- terminaltables

# Installation

```
git clone https://github.com/shakarr/scanner.git
cd scanner
```
For pip3 run:
```
pip3 install -r requirements.txt
```
For pip run:
```
pip install -r requirements.txt
```
